# Roadmap Completo

## Objetivo

Construir uma formação sólida em Ciência de Dados, começando pela base matemática e estatística, avançando depois para programação, análise exploratória, machine learning e temas mais avançados.

A ideia desta trilha não é correr para modelos complexos.
A ideia é formar uma base forte para entender o porquê das coisas e conseguir resolver problemas de forma madura.

---

## Filosofia

Esta trilha foi organizada com os seguintes princípios:

- começar pelos fundamentos
- evitar pular etapas
- construir profundidade antes de velocidade
- ligar teoria e prática constantemente
- estudar de forma sustentável

Não quero ser apenas alguém que usa ferramentas.
Quero desenvolver raciocínio analítico, pensamento estatístico e capacidade real de modelar problemas.

---

## Estrutura principal da trilha

### Fase 1 — Matemática
Objetivo:
- fortalecer raciocínio lógico e quantitativo
- criar base para estatística e machine learning

Tópicos:
- aritmética
- álgebra básica
- funções
- gráficos
- proporções
- exponenciais e logaritmos
- vetores e matrizes
- cálculo introdutório
- álgebra linear

---

### Fase 2 — Estatística
Objetivo:
- aprender a pensar com dados
- entender inferência, incerteza e variabilidade

Tópicos:
- estatística descritiva
- distribuições
- probabilidade
- amostragem
- CLT
- intervalos de confiança
- testes de hipótese
- desenho de experimentos
- A/B testing
- regressão inicial

---

### Fase 3 — Programação para dados
Objetivo:
- aplicar os conceitos usando ferramentas práticas

Tópicos:
- Python
- pandas
- numpy
- visualização
- notebooks
- SQL

---

### Fase 4 — Análise exploratória de dados
Objetivo:
- aprender a entender os dados antes de modelar

Tópicos:
- limpeza
- exploração
- visualização
- outliers
- padrões
- relações entre variáveis
- leitura crítica de bases

---

### Fase 5 — Machine Learning
Objetivo:
- entrar em modelagem com base sólida

Tópicos:
- regressão
- classificação
- métricas
- validação
- overfitting
- feature engineering
- interpretação de modelos

---

### Fase 6 — Deep Learning
Objetivo:
- estudar redes neurais com base matemática e estatística mais madura

Tópicos:
- redes neurais
- backpropagation
- embeddings
- NLP
- visão computacional

---

### Fase 7 — MLOps e produção
Objetivo:
- aprender a operacionalizar soluções de dados e IA

Tópicos:
- versionamento
- pipelines
- deploy
- monitoramento
- automação
- governança

---

## Regras de progressão

Só avançar com confiança quando houver boa compreensão dos fundamentos anteriores.

Exemplos:
- não correr para ML sem entender estatística e análise
- não estudar modelos complexos sem entender os dados
- não depender apenas de biblioteca pronta

---

## Direção de longo prazo

Quero construir uma formação que me permita:
- entender problemas complexos
- estruturar hipóteses
- analisar dados com fundamento
- testar ideias corretamente
- aplicar Ciência de Dados com maturidade
- resolver problemas reais com clareza
