## Como estudar

### Matemática
- estudar teoria
- fazer exercícios no papel
- revisar pontos fracos
- registrar dúvidas

### Estatística
- ler com calma
- resumir conceitos
- aplicar em exemplos simples
- conectar com Python

### Python e SQL
- manter contato frequente
- usar como apoio, não como foco principal
- criar notebooks curtos

---

## Regra prática

Se algum curso ou tema secundário começar a atrapalhar matemática e estatística, ele perde prioridade.
A base vem primeiro.
