# Livros e Ordem de Leitura

## Objetivo

Organizar os livros da trilha de forma clara, separando:
- leitura atual
- próximos livros
- leituras futuras

A ideia não é ler muitos livros ao mesmo tempo.
A regra será:
- 1 livro principal
- 1 livro de apoio
- próximos organizados por fase

## Regra de leitura

- não abrir vários livros pesados ao mesmo tempo
- manter foco na fase atual
- registrar resumos e avanços
- usar os livros para construir base, não para colecionar leitura
