## Critérios de evolução

### Matemática
- entender melhor funções, estrutura algébrica e raciocínio quantitativo
- ganhar segurança para avançar

### Estatística
- compreender conceitos centrais sem decorar superficialmente
- aplicar ideias em exemplos simples

### Programação
- manter contato constante com Python e SQL
- usar prática como apoio da teoria
